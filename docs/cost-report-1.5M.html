3-1. 변동비 (1.5M자 1편당)

Translation Worker Lambda

호출 수 = 575 (재시도 포함) 평균 실행 시간 = 45s (Gemini API 응답 + DB write) GB-초 = 575 × 45 × 1.536 = 39,744 GB-s 가격 (Seoul) = $0.0000166667/GB-s 비용 = 39,744 × $0.0000166667 ≈ $0.66 요청 비용 = 575 × $0.20 / 1,000,000 ≈ $0.0001 (무시)

Bible Worker Lambda

호출 수 = 115 평균 실행 시간 = 30s GB-초 = 115 × 30 × 1.024 = 3,533 GB-s 비용 = 3,533 × $0.0000166667 ≈ $0.06

SQS

메시지 수 = 575 + 115 = 690 연산 수 (send + receive + delete) = 690 × 3 ≈ 2,070 비용 = 2,070 × $0.40 / 1,000,000 ≈ $0.001

데이터 전송 (Lambda → Gemini API egress)

총 토큰 ~9.2M ≈ ~30~40 MB 페이로드 (요청+응답) egress 가격 = $0.09/GB → ≈ $0.004

CloudWatch Logs (운영 로그 모드)

평균 1MB/invocation × 690 = ~700 MB ingestion = 0.7 GB × $0.50 = $0.35 storage = $0.03/GB-월 × 0.7 GB = $0.02

6. 참고 사항

  • 재시도 보정 1.15배: CONTENT_BLOCKED, 503/overload, 빈 응답 등으로 인한 실측 평균. 안전 우회 프롬프트 동작 시 일부 호출은 2배 토큰 소비.
  • Gemini Preview 모델 가격: 공개되지 않은 경우 GA 모델(gemini-2.5-flash) 가격을 사용. 정식 출시 후 가격 재검토 필요.
  • RDS Free Tier: 신규 계정 12개월간 db.t3.micro 750h/월 무료 — 본 인프라(t4g.large)는 Free Tier 대상 아님.
  • 병렬도: Translation 동시성 50, Bible 10. 1.5M자 1편을 단일 작업으로 돌리면 wall-clock 약 30~60분 소요 예상 (실제 비용은 동시성과 무관).
  • CONTENT_BLOCKED 폴백: 안전 우회 프롬프트로 재시도 시 추가 호출 1~3회/문제챕터 발생. 메모리 기록상 64화 사례 참고.
